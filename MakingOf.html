Maintainability

To make our website maintainable we implemented the following below:

  • Correct HTML Tags Placement: Script and link tags inside the head.
  • Creating an Appropriate Title Tags.
  • Using External Style Sheets.
  • Following the standardized code style: Lowercase Markup to help with code readability.
  • Minifying CSS: removing unused code to make the code's file size smaller.
  • Using names that are understandable to humans.
  • Adding Comments to HTML, CSS and JavaScript .
  • Reducing redundancy.
